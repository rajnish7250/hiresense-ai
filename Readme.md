# 🧠 HireSense AI – Resume Analyzer

HireSense AI is an AI-powered resume analysis system that evaluates resumes against job descriptions using semantic similarity and LLM-based feedback.

## 🚀 Features

- 🔍 Resume vs Job Description Matching
- 📊 Match Score (Semantic Similarity)
- 🧠 AI Feedback (LLM-based + fallback)
- 📌 Skill Gap Analysis
- ⚡ Optimized with caching

## 🏗️ Architecture

- UI: Streamlit
- Controller Layer
- Service Layer (Pipeline)
- Core (Embeddings, Similarity, NLP)

## 🛠️ Tech Stack

- Python
- Streamlit
- SentenceTransformers
- NumPy
- OpenAI (optional)
